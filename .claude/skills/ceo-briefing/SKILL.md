# CEO Briefing Skill

**Purpose**: Generate the Monday Morning CEO Briefing with full business intelligence
**Tier**: Gold
**Output**: `AI_Employee_Vault/Briefings/YYYY-MM-DD_Monday_Briefing.md`
**Automated**: Runs every Sunday at 20:00 via Gold orchestrator

---

## When to Use

- User asks "generate my weekly briefing"
- Monday morning review
- End-of-week business summary needed
- Investor/partner update preparation
- Manual trigger: "run weekly audit"

---

## What the Briefing Contains

```
# Monday Morning CEO Briefing
Generated: {timestamp}
Period: {last 7 days}

## Executive Summary
{1-2 sentence overall assessment}

## Revenue & Financials
| Income | Expenses | Net | MTD% |

## Completed Tasks This Week
- [x] task1 — timestamp
- [x] task2 — timestamp

## Social Media Performance
- Facebook: N posts, N likes
- Instagram: N posts, N followers
- Twitter: N tweets, N likes

## System Health
| Total Actions | Errors | Error Rate |

## Subscription Audit
(flags unused or overpriced subscriptions)

## Active Projects
(from Business_Goals.md)

## Items Pending Approval
(from Pending_Approval/ folder)
```

## Manual Generation

### Option A — Run the Python script directly
```bash
cd Gold-Tier
venv\Scripts\activate
python orchestrator\weekly_audit.py
```

### Option B — Use the Python function
```python
from orchestrator.weekly_audit import generate_ceo_briefing
briefing_path = generate_ceo_briefing()
print(f"Briefing: {briefing_path}")
```

### Option C — Via Claude (this skill)

1. Read `Business_Goals.md` for targets
2. List `Done/` for completed tasks this week
3. Read `Accounting/transactions_*.json` for financials
4. Read `Social_Media/*.md` for engagement stats
5. Read `Logs/*.json` for system health
6. Generate briefing using the template below
7. Write to `Briefings/{date}_Monday_Briefing.md`
8. Log to audit: `audit_log('manual_ceo_briefing', actor='claude')`

## Data Sources

| Section | Source | Fallback |
|---------|--------|---------|
| Revenue | `Accounting/transactions_*.json` | "No data" |
| Tasks | `Done/*.md` (last 7 days by mtime) | Empty list |
| Social | `Social_Media/*.md` | "No reports" |
| Health | `Logs/*.json` (last 7 days) | "No logs" |
| Goals | `Business_Goals.md` | Default targets |
| Pending | `Pending_Approval/*.md` | "All clear" |

---

## Business Goals Setup

Edit `Business_Goals.md` to customize targets:
```markdown
### Revenue Target
- Monthly goal: Rs 100,000  ← update this
```

The briefing will calculate MTD percentage automatically.
